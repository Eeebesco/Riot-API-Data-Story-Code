# Riot-API-Data-Story-Code
Codes for the scripts to retrieve match histories and match data for a user.
match_history_retriever.py is used first to retrieve the specified amount of matches of a user via their puuid
match_data_retriever_final.py is used after to retrieve the specified match data from the already retrieved match IDs.
Both codes can be edited to get a different number of games or different statistics for the game. Look at the Riot API for data titles.
match_data_retriever_v1.py can be used similar to the final version of the code to retrieve the information of all players in the game,
not just the specified player.
